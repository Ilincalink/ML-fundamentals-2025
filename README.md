 Project 1: Classification Problem - Titanic Survival Prediction
Problem Statement
Predict passenger survival on the Titanic using demographic and travel information.

Dataset Characteristics
Type: Historical passenger data

Source: Public dataset (Excel format)

Target: Binary classification (survival)

Key Features: Passenger class, gender, age, fare, family relations

Challenges: Missing values, categorical variables, imbalanced classes

Technical Focus Areas
Handling missing data and imputation

Categorical variable encoding techniques

Feature scaling and normalization

Addressing class imbalance

Feature selection methods

Logistic regression implementation

 Project 2: Regression Problem - Bike Sharing Demand Prediction
Problem Statement
Predict hourly bike rental counts based on environmental, seasonal, and temporal factors.

Dataset Characteristics
Type: Time-series transportation data

Source: UCI Machine Learning Repository

Target: Continuous value (rental count)

Key Features: Weather conditions, temporal patterns, seasonal factors

Records: ~17,000 hourly observations

Technical Focus Areas
Temporal feature engineering

Working with weather and seasonal data

Implementing multiple regression algorithms

Hyperparameter tuning and optimization

Model performance comparison

Bias-variance tradeoff analysis
