# MachineLearning-Fundamentals-2025

## Repository Overview
This repository contains **two comprehensive machine learning projects** that demonstrate **end-to-end data science workflows**, from data preprocessing and feature engineering to model development and evaluation.

The projects focus on applying machine learning techniques to **real-world problems** using structured and time-series data.

- **Model Evaluation & Selection**
  - Performance metric selection

## Project 1: Classification – Titanic Survival Prediction

### Problem Statement
Predict whether a passenger survived the Titanic disaster using demographic and travel-related information.

### Dataset Characteristics
- **Type:** Historical passenger data  
- **Source:** Public dataset (Excel format)  
- **Target Variable:** Survival (Binary classification)  
- **Key Features:**
  - Passenger class
  - Gender
  - Age
  - Fare
  - Family relations (SibSp, Parch)

### Challenges
- Missing values
- Categorical variables
- Class imbalance

### Technical Focus Areas
- Missing value handling and imputation
- Categorical variable encoding
- Feature scaling and normalization
- Addressing class imbalance
- Feature selection techniques
- Logistic Regression implementation

## Project 2: Regression – Bike Sharing Demand Prediction

### Problem Statement
Predict **hourly bike rental demand** based on environmental, seasonal, and temporal factors.

### Dataset Characteristics
- **Type:** Time-series transportation data  
- **Source:** UCI Machine Learning Repository  
- **Target Variable:** Rental count (Continuous value)  
- **Records:** ~17,000 hourly observations  
- **Key Features:**
  - Weather conditions
  - Seasonal indicators
  - Temporal features (hour, day, month)

### Technical Focus Areas
- Temporal feature engineering
- Working with weather and seasonal data
- Multiple regression algorithms
- Hyperparameter tuning and optimization
- Model performance comparison
- Bias–variance tradeoff analysis

## Project Workflow
Each project follows a systematic machine learning pipeline:

1. **Problem Understanding**
   - Define objectives and success criteria

   - Data quality assessment

3. **Data Preprocessing**

   - Feature engineering
   - Train-test-validation split

4. **Model Development**
   - Algorithm selection
   - Training and validation
   - Hyperparameter tuning

5. **Evaluation & Deployment**

   - Model selection
